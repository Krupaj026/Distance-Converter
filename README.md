# Distance Converter 🌸

A simple and interactive distance converter built using **HTML, CSS, and JavaScript**. The project converts **miles to kilometers** and **kilometers to miles** dynamically. Users can switch between conversion modes using keyboard keys, creating a more interactive experience.

## ✨ Features

* Convert Miles → Kilometers
* Convert Kilometers → Miles
* Keyboard shortcuts for switching modes
* Dynamic DOM updates using JavaScript
* Cute pastel themed responsive UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ⌨️ Controls

* Press **K** → Kilometers to Miles
* Press **M** → Miles to Kilometers
